## Function
자바스크립트의 가장 기본적인 기능은 **HTML을 조작**하는 것이다.   
HTML 태그의 속성값을 바꾸거나, 태그를 수정, 삭제 할 수 있다.

## Characteristic
* scripting language이다. (runtime에 interpreter에 의해 실행된다.)
* 싱글 스레드 환경에서 실행된다.     
* [비동기적]()으로 실행된다.

## NodeJS
JavaScript가 처음 등장했을 때에는 웹 브라우저에서만 실행 될 수 있었다.   
자바스크립트는 웹 브라우저의 JS엔진에 의해 실행된다.
